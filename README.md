# 📊 Call Center Data Analysis Dashboard (Excel)

## 📌 Project Overview
This project analyzes customer call center operations using Microsoft Excel.  
The objective is to transform raw call center data into meaningful insights using Pivot Tables, charts, and an interactive dashboard.

The dashboard helps visualize:
- Call volume trends
- Customer satisfaction ratings
- Representative performance
- Revenue contribution
- Call distribution across cities

---

## 🎯 Business Objective
Call centers generate large amounts of operational data, making it difficult to quickly understand performance trends.

This project aims to answer key business questions:

- How does call volume change over time?
- Which representative handles the most calls?
- Which city generates the highest call volume?
- How satisfied are customers with the service?
- How much revenue is generated from calls?

---

## 📂 Dataset Information
The dataset contains **1000 call records** including customer and operational details.

### Dataset Features
- Customer ID
- City
- Gender
- Call Representative (R01 – R05)
- Call Duration
- Call Amount
- Customer Rating
- Call Date

## 📈 Key Dashboard Visualizations

### 1. Monthly Call Trend
Shows how call volume changes across months.

Insight:
- Peak call volume occurs during **March and April**
- Call activity declines toward the **end of the year**

---

### 2. Calls by Day of Week
Displays which days receive the highest number of calls.

Insight:
- **Weekends receive more calls**
- **Tuesday and Thursday have lower call volumes**

---

### 3. Female vs Male Callers
Compares gender distribution of callers across cities.

Cities analyzed:
- Cincinnati
- Cleveland
- Columbus

---

### 4. Customer Rating Distribution
Shows how customers rate their call experience.

Insight:
- Most customers provide **ratings of 4 or 5**
- Overall customer satisfaction is **high**

---

### 5. Representative Performance
Displays the number of calls handled by each representative.

| Representative | Calls |
|---|---|
| R01 | 189 |
| R02 | 218 |
| R03 | 207 |
| R04 | 186 |
| R05 | 200 |

Insight:
- **R02 handled the highest number of calls**

---

### 6. Call Volume by City
Shows distribution of calls across cities.

Insight:
- **Cleveland has the highest call volume**

---

## 📊 Key Insights
- Customer satisfaction is generally **positive**
- Call demand varies significantly across cities
- Some representatives handle **more calls than others**
- Revenue contribution differs across representatives
